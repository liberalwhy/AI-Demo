# 示例输入

你可以在 Streamlit 页面中复制以下内容进行测试。

## 商品名称
便携式榨汁杯

## 商品品类
小家电 / 厨房电器 / 便携健康饮品

## 核心卖点
无线便携；快速榨汁；易清洗；适合办公室和宿舍；轻量不占空间

## 目标用户
上班族、健身人群、学生、租房人群

## 售卖平台
小红书 / 抖音 / 淘宝

## 内容目标
种草转化，降低用户对便携榨汁杯闲置和清洗麻烦的顾虑

## 价格区间
99-199 元

## 使用场景
办公室早餐、宿舍夜宵、健身后补充水果饮品、外出旅行

## 禁忌/不能夸大的点
不能宣称减肥、治疗、排毒、医学功效；不能说 100% 不漏水；不能贬低竞品

## 参考风格
小红书真实测评风，生活化、轻松、可信，不要过度广告感

## API 模式测试建议

1. 先用“本地模板模式”确认项目正常运行。
2. 再切换到“大模型 API 模式”。
3. 填写 API Key、Base URL、模型名称后点击生成。
4. 如果 API 失败，页面会自动切回本地模板模式作为兜底。
